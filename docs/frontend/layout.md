# 布局系统

## AdminLayout

管理端使用统一的 `AdminLayout` 布局组件，位于 `layouts/AdminLayout.vue`。

## 布局结构

```
┌──────────────────────────────────────────────┐
│  顶部导航栏（Header）                          │
│  ┌──────────┬──────────────────────────────┐ │
│  │ Logo     │ 面包屑         用户菜单      │ │
│  └──────────┴──────────────────────────────┘ │
├──────────┬───────────────────────────────────┤
│          │                                   │
│  侧边栏   │  内容区域                         │
│ (菜单)   │  ┌─────────────────────────────┐ │
│          │  │  <RouterView />             │ │
│  · 仪表盘 │  │                             │ │
│  · 系统   │  │  动态路由渲染的页面           │ │
│    · 用户 │  │                             │ │
│    · 角色 │  │                             │ │
│    · ... │  │                             │ │
│  · 权限  │  │                             │ │
│  · 文件  │  │                             │ │
│          │  └─────────────────────────────┘ │
│          │                                   │
└──────────┴───────────────────────────────────┘
```

## 主要功能

### 侧边栏菜单

- 从 `authStore.menuTree` 动态生成
- 支持多级菜单嵌套
- 可折叠/展开
- 图标显示
- 当前路由高亮

### 顶部导航栏

- Logo 和标题
- 面包屑导航（自动根据当前路由生成）
- 侧边栏折叠按钮
- 用户信息下拉菜单
  - 个人信息
  - 修改密码
  - 退出登录

### 内容区域

- `<RouterView />` 渲染当前路由对应的页面组件
- 自适应宽度和高度

## 响应式

- 侧边栏可通过按钮折叠为图标模式
- 内容区域自适应剩余宽度
